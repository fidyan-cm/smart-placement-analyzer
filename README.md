# Smart Placement & Skill Gap Analyzer

An AI-powered Django web application that predicts student placement probability using Machine Learning and identifies skill gaps.

## Features
- Placement prediction using Logistic Regression, Random Forest, and XGBoost
- Skill gap analysis with industry benchmarks
- Interactive dashboard with Gauge and Radar charts
- HackerRank profile integration
- PDF report download
- User authentication

## Tech Stack
- Python, Django, Scikit-Learn, XGBoost
- Chart.js, Bootstrap 5
- WeasyPrint for PDF generation
- Deployed on Railway
